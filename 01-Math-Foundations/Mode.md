# Mode:

Values that are most frequently lies in dataset.

Example : 2,3,3,4,5,3
Mode = 3 , ( 3 appears 3 times , highest occurance)

# Mode works / use:

Mode works best for categorical data , mean and median work for numerical 
 Example: 
 Customer daily buying rank Fruits :  Apple , Mango , Apple, Banana , Apple , Watermelon

 Mode = Apple ,
 Its the categorical data
 We can say Apple is most leikely fruit for that area people , that is getting hgher purchase daily.

 # Type of Mode 

 1. Unimode - one value occurs most
 2. Bimodal - Two value occurs most
 3. Multimodal - Multiple value occurs most

 # No mode:
 If all value occur only one time , then that is no mode. mode = NIL

 # Usage
 - Mode Mostly used in Classification problems
 - If catgeorical data is missing we filled with mode of the column
 - Voting  classifiers
 - Random forest Majority Voting.
